        # EAM势函数电荷密度度量规则

        本章节不含大量的代码，更多的是对于理论进行更深一步地探索

        # 1. Zhou势的对势部分分析

        Zhou使用了Wadley修改的对势函数，其在r趋近于0（甚至r直接取0在数学上也是有意义的），其极限值是一个有限的值，接近 Aeα−Beβ ，所以Zhou的对势在图像上，在接近于r为0的部分，并不会向着无穷大趋近。

        ψ(r)=Ae−α(r/re−1)1+(r/re−κ)20−Be−β(r/re−1)1+(r/re−λ)20

        而如果我们分析Zhou引用的嵌入能论文，就会发现，Zhou势中所使用的嵌入能，在原子间距较小时，便能很好地描述原子体系的总能量。所以这时的对势接近一个有限的常数，就不会干扰到总能量的计算。

        # 2. UBER理论模型

        UNIVERSAL BINDING-ENERGY RELATION (UBER)这个理论模型是很早提出的，大致意思是对于能量和特质长度进行一定的scaled之后，大多数材料的scaled energy和scaled length都符合同一个方程。当然，这个模型也是存在很多局限的，比如模型不能发生剪切（改变晶格形状）

        Wadley引用的嵌入能就来自于一篇证明UBER这一模型的文章(doi: 10.1103/physrevb.37.6632)，而这篇文章中的证明对象是用金属的电子气模型来证明的（但是并非自由电子气，加入了一些对于交换关联能的估计）。文章中给出的E(n)说明了在这一模型下，总能量E应该是电荷密度的函数，并且给出了具体的模型形式，最终指出这个模型在平衡态以及电荷密度较高的部分适用。模型的具体形式和Zhou论文中嵌入能的第三个分段函数一致。

        所以原子间距较小，电荷密度较高的位置，更多是以嵌入能来估计系统的总能量。（这个观点是我的个人猜测）

        # 3. 电荷密度的估计

        在拥有对于上述两个背景的理解之后，我们再去查看Wadley的论文时，就会发现他是以[能量]/[长度]为量纲作为电荷密度的。我尝试了很多猜测去从DFT中计算出这一物理量（其中最接近的是用最高费米能级乘对应体系的费米波矢长度，但是仅仅对Ni有效）

        后来，在重新阅读Wadley的论文后（49页那篇），我发现他们对于ρ(r)=feBe−β(r/re−1)1+(r/re−λ)20 中的 fe 的拟合是一种经验公式去拟合的 fe=Ecoh/Ω1/3 这不仅解释了量纲问题，而且Wadley的观点是，这个 fe 在用于对纯金属的拟合中是没有任何作用的，其作用在于后续和合金对齐（也是一种scaled的想法）

        那么平衡态电子密度 ρe 就不是直接通过VASP之类的第一性原理计算软件计算出来的，而是直接使用EAM模型计算出来的，因为 ρe 和 fe 需要拥有相同的物理意义，所以他们两者的量纲均为[能量]/[长度]

        再补充一句，因为考虑电子气模型在金属中的能量估算公式为 E(n)=An2/3−Bn1/3 ，所以除体积的三分之一次方也不是乱取的，因为Zhou势中嵌入能的前两段分段是三次函数。最后，我还认为Wadley这种scaled的主导思路很偏向于传统物理的基调，即给出一个适用于诸多体系的理论模型（这也是为啥他只用单元素拟合合金），而通过多元素合金来进一步拟合，其实是更偏向于一种数据驱动的哲学。
