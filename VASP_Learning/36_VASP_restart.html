        # VASP 重新开始探索

        # 1. 初期探索阶段问题总结

        对于初期的工作流设计，出现了以下4个在物理理论上的不合理之处，分别是：

        1. For elemental configuration, the 3×3×3 supercell is unnecessary.

        2. The relaxed structure is estimated, and the structure optimization in VASP is necessary for alloy. Even the re-relax calculation should be taken.

        3. The stress of the configuration should be calculated.

        4. The strain is repeated for DeePMD-kit’s dataset. Because the DeePMD-kit’s descriptor is set for rotation, we only need to calculate the strain in one axis.
