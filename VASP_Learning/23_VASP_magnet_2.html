    # VASP高精度微调数据-3

    对于一元金属计算时，我们之前进行计算和模拟时，对于relaxed构型的晶体结构是通过得到能量最低值来确定的构型。

    所以我们对于估计的relaxed构型晶体结构，在打开磁性的基础上，进行高精度的结构优化。

    # 1. 高精度结构优化

    首先，并不是所有元素的基态均为FCC结构，但是我们这里在计算时，以之前FCC构型下的能量最小的构型作为输入POSCAR

    # 1.1 INCAR脚本

    使用下面脚本进行优化

    IBRION = 2

NPAR = 8

ISMEAR = 1
SIGMA = 0.2

ISPIN = 2

ISTART = 0
ICHARG = 2

ISIF = 3
ISYM = 0

NSW = 1000
NELM = 400

PREC = Normal

EDIFF = 1E-4

ALGO = Fast

KSPACING = 0.2
LWAVE = .FALSE.
LCHARG  =  .FALSE.

    # 附录1 收敛性检查脚本

    在进行高精度计算的过程中，往往存在数据不收敛的情况，因此给出检查数据收敛性脚本如下所示。

    （目前体感下来不收敛的数据大约是五十分之一，比不开自旋的万分之一的比例还是高很多的，而且我的电子步上限至少200，普遍400）

    #!/bin/bash

for dir in */;
do
    cd $dir
    echo $dir
    grep "unconverged" OUTCAR
    cd ..
done
