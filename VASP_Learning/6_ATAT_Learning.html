          # ATAT初步学习

          合金理论自动化工具包(ATAT)是一个通用名称，指的是 Axel van de Walle与多个研究小组合作开发的工具。

          # 1 下载工具包ATAT

          在下面的官网中下载源码：

          http://brown.edu/Departments/Engineering/Labs/avdw//atat/atat3_36.tar.gz

          tar -xvf atat3_36.tar.gz
#说明：压缩文件没有用gzip格式压缩，因此不需要用选项-z
          1
          2

          修改makefile文件，由于我已经在服务器上安装了Intel工具包，所以将MPICXX编译器改为intel的c++并行编译器。

          修改的makefile文件如下

          BINDIR=/home/ifxie/06_atat/atat/bin
#for g++ compiler on all platforms
CXX=g++

#for MPI: limited implementation at this point
MPICXX=mpicxx -DATAT_MPI 

all:
	./foolproof.sh ${BINDIR}   "${CXX}"
	${MAKE} -C src clean
	${MAKE} -C src          "CXX=${CXX}"

mpi:
	./foolproof.sh ${BINDIR}   "${MPICXX}"
	${MAKE} -C src clean
	${MAKE} -C src          "CXX=${MPICXX}"  mpi

force:
	${MAKE} -C src          "CXX=${CXX}"

clean:
	${MAKE} -C src clean

install:
	echo set atatdir="$(PWD)" > $(HOME)/.atat.rc
	${MAKE} -C src          BINDIR=${BINDIR} install
	${MAKE} -C glue/jobctrl BINDIR=${BINDIR} install
	${MAKE} -C glue/vasp    BINDIR=${BINDIR} install

uninstall:
	rm -f `cat src/safecpdir/touninstall.txt glue/jobctrl/safecpdir/touninstall.txt glue/vasp/safecpdir/touninstall.txt`
	rm -i ${HOME}/.atat.rc ${HOME}/.ezvasp ${HOME}/.machines.rc

          # 2 安装sqs2poscar

          sqs2poscar是一个基于C++代码写的插件，作用是“Converting output from ATAT mcsqs to POSCAR format”。

          git clone https://github.com/c-niu/sqs2poscar.git
cd sqs2poscar
vi sqs2poscar.cpp
          1
          2
          3

          在C++代码前加上两个库（原因可以自己看插件的Readme）

          #include <cstring>
#include <stdlib.h>
          1
          2

          用c++编译器编译

          c++ ./sqs2poscar.cpp -o ./sqs2poscar
          1

          配置环境变量

          vi ~/.bashrc
export PATH=/home/lebedmi2/SOFTWARE/ATAT/sqs2poscar:$PATH
source ~/.bashrc

          运行mcsqs命令

          mcsqs -2=2 
mcsqs -rc
          1
          2

          一般可以提前停止，否则整个运行时长极长，然后使用bestsqs.out文件是自由能最低的sqs；

          可以在mcsqs.log文件中查询自由能。

          用插件对于输出结果进行转换。

          sqs2poscar bestsqs.out
          1

          设置晶格常数。

          sed -i '2s/.*/3.45/' bestsqs.out-POSCAR
          1

          说明：可以修改部分有1个，晶格常数3.45,注意这里不是超晶胞的晶格常数，而是原胞的晶格常数。

          最后，用atomsk对于文件bestsqs.out-POSCAR进行优化

          atomsk bestsqs.out-POSCAR -sort species up -frac POSCAR
