    # VASP安装教程

    VASP(Vienna Ab initio Simulation Package)，是一款维也纳大学研发的对于分子结构等的基于第一性原理的计算软件。

    # 1. VASP环境安装

    计算环境，是一款科学计算软件在运行时，需要使用的编译器，科学计算库。而VASP是基于Fortran为源码编写，运用了MPI并行协议，调用了部分科学计算库，因此我们使用VASP时，需要将对应的编译器，计算库安装在计算机上。

    # 1.1 安装Intel® Parallel Studio XE 2020

    推荐安装的理由是，其一，对于我们服务器的硬件，Intel编译器效果会更好；其二，Intel编译器能够满足VASP所需的环境要求：

    ·Compilers for Fortran (at least F2008 compliant), C, and C++.

    ·Numerical libraries: FFTW, BLAS, LAPACK, and ScaLAPACK.

    ·An implementation of the Message Passing Interface (MPI).

    只要能够弄到intel编译器的安装包及Licenses文件即可简单地按照其内容安装了。而弄到的方法是很多的，而且张寅老师肯定是有的。

    # 2. VASP安装

    VASP的安装包可以直接找张寅老师要，顺便要赝势文件。找我要的话只有6.4.1版本以及当时的赝势文件

    # 2.1 VASP安装

    解压后进入VASP中的arch目录复制该目录下的makefile.include.linux_intel文件到上一级目录，并改名为makefile.include。这一步是确定makefile的include头，而因为我们之前安装的环境是intel编译器所以选择这个include头。如果你是手搓的环境，选makefile.include.gnu_omp

    然后对于这个makefile.include文件再稍作调整。调整结果如下

    然后确认自己的Intel编译器的环境是打开了的

    source ~/intel/oneapi/setvars.sh
    1

    然后即可执行VASP的安装，可以根据需求安装std,gam,ncl版本，每个版本安装时间约为10分钟。

    make all
#一次性安装上述3个版本
    1
    2

    安装完成后，~/vasp.6.4.1/bin/vasp_std即为可执行程序，用下面命令调用。

    mpirun -np 8 ~/vasp.6.4.1/bin/vasp_std
    1

    # 3. VASP插件说明

    VASP的插件生态极为丰富，以及其他上下游计算软件生态也十分丰富，几乎都会做格式对齐。这里简要记录张老师提过的建议：

    1. 如果是处理格式转化的问题，用atomsk软件或ASE库；

    2. 如果是处理文件位置，调用程序，则推荐使用bash脚本；

    3. 如果在2的基础上设计复杂的数据处理工作，则推荐python脚本。

    以及下面是我个人补充：

    1. 如果是涉及合金的生成，则推荐使用ATAT。

    上述插件的安装和使用都是极为便捷的，网络上可学习资料较多，这里不做详细说明。
