    # 电荷密度计算

    在EAM计算过程中，需要使用平衡构型中删除一个原子时的ρe作为一个初始参数进行输入。

    通过VASP获得电荷密度ρe，以及对于ρ的量级有一个大概认知，是本篇工作的内容

    # 1. CHGCAR文件

    文件的组成：开头部分和POSCAR文件一致。中间会给一个FFT-grid的NGXF, NGYF, NGZF。然后就是按照下面这一方式将每个网格点上的电荷密度n(r)输出出来：

    WRITE(IU,FORM) (((C(NX,NY,NZ),NX=1,NGXF),NY=1,NGYF),NZ=1,NGZF)
#vasp的fortran源码：最外循环为z，中间循环为y，最内循环为x，所以x是变化最快的，其次是y，最慢的是z。
    1
    2

    最后再给出Augmentation occupancies说明轨道占据情况。

    但是如果增加了电子的自旋运算的话，则还会重复地给出一个FFT-grid，然后给出每个网格点上的自旋密度。最后再给出Augmentation occupancies说明轨道占据情况。

    总电荷密度与自旋密度：总电荷密度是spinup+spindown，而自旋电荷密度是spinup−spindown，所以我们在计算EAM的电荷密度的时候，后者的数值对于我们意义不大，前者的数值才是我们想要的。

    # 1.1 总电荷密度的单位分析

    定义CHGCAR中r位置处的总电荷为data(r)。FFT-grid对应的NGXF, NGYF, NGZF在文件中也存在，晶胞的体积取POSCAR的2-5行计算出来为Vcell。

    那么电荷密度n(r)=data(r)/[Vcell×(NGXF×NGYF×NGXF)]，单位为 1/\AA3

    # 1.2 FFT-grid与实空间的关系

    r(Nx,Ny,Nz)=Nx−1NGXFa+Ny−1NGYFb+Nz−1NGZFc

    根据这一关系，如果我们将(0,0,0)处的原子取走，那么CHGCAR中的第一个data(r)对应就就是这个空位的电荷密度，也是我们EAM中希望进行拟合的电荷密度。

    # 2. 计算空位晶格电荷

    # 2.1 POSCAR建模

    首先先建立一个3×3×3的超晶胞，晶格常数与基态计算结果保持一致。使用超晶胞计算的原因是希望两个空位之间隔开的距离是大于截断半径的。

    然后将（0，0，0）处的原子对应的行从POSCAR中删除，并将总原子数减1（很容易忘！）。

    其他输入文件在附录中。

    # 2.2 计算结果

    计算得到的在(0,0,0)处的局域电荷密度为7.847×10−9e/\AA−3

    IBRION = 2             (Structure optimization)

    ICHARG = 2             (Non-self-consistent: GGA/LDA band structures)
    LREAL  = F             (Projection operators: automatic)
    ENCUT  =  500          (Cut-off energy for plane wave basis set, in eV) 
    PREC   =  Accurate     (Precision level)

    LCHARG = .TRUE.        (Write CHGCAR or not) 
    ADDGRID= .TRUE.        (Increase grid; helps GGA convergence)

    EDIFF  =  1E-05        (SCF energy convergence; in eV)
    
           
Ionic Relaxation
    NSW = 0                (Max electronic SCF steps)
    ISIF   =  2            (Stress/relaxation: 2-Ions, 3-Shape/Ions/V, 4-Shape/Ions)
