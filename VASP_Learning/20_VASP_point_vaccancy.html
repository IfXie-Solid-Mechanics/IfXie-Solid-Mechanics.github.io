    # VASP五元合金点空缺构型计算工作流

    对于CrMnFeCoNi和CrFeCoNiPd高熵合金，先选出之前五元合金的relaxed构型（总计2个），对这两个构型进行原子的删除（总计108个原子位置可以被删除）。

    综上可以收集216种构型，将这216种构型放在aimd下运行400步，之后以40步的步长取出10个snapshots进行结构优化计算，最终得到2160个数据样本。

    从第9行到第116行中每一行均可以被删除，但是如果删除的行数是[9-30]时，需要将Cr原子的数量减少一个。以此类推，[31-52]行减少一个Fe原子，[53-74]行减少一个Co原子， [75-95]行减少一个Ni原子，[96-116]行减少一个Pd原子。

    # 1.2 bash脚本构建思路

    首先POSCAR的第7行原子数需要修改，修改规则见1.1末尾。其次POSCAR的第[9-116]行中某一行原子在每次操作中会被删除，并且保证删除后对应位置不出现“空行”。

    其次，明确工作目录为~/05_atomsk/point_vaccancy_wuzhen_1/CrFeCoNiPd，先将CrFeCoNiPd的relaxed构型的POSCAR文件和POTCAR文件复制到这一个目录下：

    #!/bin/bash

for line_number in $(seq 9 1 116)
do
    num=$(($line_number-8))
    
    mkdir CrFeCoNiPd_$(printf "%03d" $num)
    cd CrFeCoNiPd_$(printf "%03d" $num)
    cp ../POSCAR .
    cp ../POTCAR .
    
    # 使用sed命令删除指定行
    sed -i "$line_number d" POSCAR

    # 使用sed命令修改第7行原子数：ge是大于或等于，gt才是严格大于
    if [ $num -ge 1 ] && [ $num -le 22 ]; then
        sed -i "7 s/.*/21      22      22      21      21/" POSCAR
    elif [ $num -ge 23 ] && [ $num -le 44 ]; then
        sed -i "7 s/.*/22      21      22      21      21/" POSCAR
    elif [ $num -ge 45 ] && [ $num -le 66 ]; then
        sed -i "7 s/.*/22      22      21      21      21/" POSCAR
    elif [ $num -ge 67 ] && [ $num -le 87 ]; then
        sed -i "7 s/.*/22      22      22      20      21/" POSCAR
    elif [ $num -ge 88 ] && [ $num -le 108 ]; then
        sed -i "7 s/.*/22      22      22      21      20/" POSCAR
    fi
    cd ..
done
