# VASP五元合金的大变形 & VASP高精度微调数据-2

# 1. 计算思路

本篇工作主要聚集与采样大变形的数据样本，同时绘制应力应变曲线，分析线性段的范围，对于后续的测量杨氏模量的工作有指导意义。

# 1.1 输入文件生成脚本

用结构优化前的估计relaxed构型，先在x方向做单轴拉伸构型，应变从-0.3到0.3之间，步长为0.01。

第一个脚本是用于检查文件是否正确生成。

#!/bin/bash

mkdir tmp
atomsk POSCAR tmp.lmp
mv tmp.lmp ./tmp/

for i in $(seq 1 1 60);
do
    #le:less than or equal
    if [ $i -le 30 ]; then
            ii=$(echo "scale=2; $i/100" | bc)
    else
            tmp=$(($i - 30))
            ii=$(echo "scale=2; $tmp/-100" | bc)
    fi
    
    cd tmp/
    atomsk tmp.lmp -sort species up -frac -deform x $ii POSCAR
    mv POSCAR ../POSCAR-$(printf "%03d" $i)
    cd ..
done

rm -r tmp/

对于CrFeCoNiPd高熵合金，大变形的应力应变曲线如下所示（由于初始构型只是估计relaxed构型，所以存在初始应力不为0的现象，换用高精度结构优化后的构型则不存在这一问题）

# 2.2 弹性模量

总结：从实验结果中可以看出，两种合金的线性区域都是在百分之5以前，所以为了更加精准地测量弹性常数，需要在更小的应变中采用更小的步长来实现。

首先，从CrFeCoNiPd-001到CrFeCoNiPd-030文件夹内，分别表示应变从0.01增长到0.30。然后提取应力的bash命令为：

综上所述，提取脚本如下所示（注意VASP的单位是kBar，因此用0.1Gpa=1Kbar进行单位换算）：

#!/bin/bash
echo "Strain    Stress_Gpa" >> SS.dat
for i in $(seq 60 -1 31);
do
        tmp=$(($i - 30))
        strain=$(echo "scale=2; $tmp/-100" | bc)
        st_x=$(grep 'in kB' CrFeCoNiPd-$(printf "%03d" $i)/OUTCAR | awk '{printf "%12.8f \n", $3}')
        stress_x=$(echo `echo "scale=3;-0.1*$st_x" | bc |awk '{printf "%.3f",$0}'`)
        echo "$strain    $stress_x" >>  SS.dat
done

for i in $(seq 1 1 30)
do
    strain=$(echo "scale=2; $i/100" | bc)
        st_x=$(grep 'in kB' CrFeCoNiPd-$(printf "%03d" $i)/OUTCAR | awk '{printf "%12.8f \n", $3}')
        stress_x=$(echo `echo "scale=3;-0.1*$st_x" | bc |awk '{printf "%.3f",$0}'`)
        echo "$strain    $stress_x" >>  SS.dat
done

import numpy as np
import matplotlib.pyplot as plt

# 读取文件数据（假设文件名为 SS.dat）
data = np.genfromtxt("SS.dat", delimiter=None, skip_header=1)
strain = data[:, 0]    
stress = data[:, 1]    

# 绘制应力-应变曲线
plt.figure()
plt.plot(strain, stress, 'b-', label='Stress-Strain Curve')
plt.xlabel('Strain')      
plt.ylabel('Stress (GPa)') 
plt.title('Stress vs. Strain')    
plt.grid(True)               
plt.legend()                 

plt.savefig("stress_strain_curve.png")
