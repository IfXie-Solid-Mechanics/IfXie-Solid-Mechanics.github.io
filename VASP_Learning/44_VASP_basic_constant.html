      # 探索晶格常数的计算方法

      # 1. 基于状态方程的思想

      首先，计算被分为了两个部分，第一部分是优化计算，第二部分是静态计算。这种方法适用于探索平衡态的晶格常数，其计算结果仅在平衡态附近较为可信。计算代码在下面附录1中

      备注，一般VASP中对于IMSEAR的选取规则：

      金属： ISMEAR= 1 or 2, SIGMA = 0.2;
半导体、绝缘体（体系不大）： ISMEAR = -5  
半导体、绝缘体（体系较大）： ISMEAR = 0， SIGMA = 0.03-0.05
未知体系： ISMEAR=0 ， SIGMA = 0.03-0.05
静态计算DOS、无论什么体系： ISMEAR=-5
声子计算（金属）： ISMEAR = 1
声子计算（半导体、绝缘体）： ISMEAR = 0 或 ISMEAR = -5

      计算结果如图所示，在3.52埃总能量最低，为-21.871787940eV,曲线也较为平滑。

      # 1.4 拓展分析-cohesive energy（内聚能）

      基态构型除了获取晶格常数，还可以获取内聚能的物理参数。但是我们观测上述计算结果，发现Ni的Total-energy计算结果-5.47eV/atom略低于Mishin计算出的-4.45eV/atom

      主要原因是我们上述计算的能量不是内聚能。由于VASP中，孤立原子的能量并不是0，然而内聚能的计算定义是 Ecoh=Ebulk−Eiso，等式右边第一项是我们已经计算出来的能量；等式右边第二项在VASP中非0，但是可以通过计算孤立原子的能量得出。

      具体孤立原子的计算方法：设计单个原子位于原点，然后设计一个极大的box，进行静态计算，具体代码在附录2中。参考了VASP官网(https://www.vasp.at/wiki/index.php/Calculation_of_atoms)。如果想要得到电子熵极小的构型，需要设置很小的SIGMA，但是SIGMA越小，收敛的难度越大。经过测试在0.05-0.1之间可以稳定的收敛（100步左右）。结果中的EENTRO通常有0.1ev的量级，所以结果不取F，取E。（这些都在官网的教程中有写）

      经过计算，我们发现单个Ni原子有能量约为-0.55028049eV，因此我们计算出的Ni的cohesive energy是-4.92eV/atom。

      # 1.4.1 内聚能误差初步分析

      理论上，内聚能是一个相对能量，即将一个原子从bulk中移除成为孤立原子所需要的能量Ecoh=Ebulk−Eiso。那么无论是什么计算软件，cohesive energy都应该是同一个数值。分析的误差原因如下：

      1. 晶格常数没有完全对上：虽然我取的晶格常数和Mishin在小数点后3位不匹配，但是理论这点误差不会对能量有-0.47eV/atom的影响。

      2. 磁矩问题，我在3.52埃计算出的总磁矩是mag=2.4195μB，并且每个原子的状态相同，呈现正常的铁磁性。而且magnet=0.605μB/atom,和0.591的磁矩也相差无几。孤立状态的Ni原子也呈现出了饱和的mag=1.999μB，符合泡利不相容原理。

      3. 精度问题：首先是POTCAR方面，选用的是经典POTCAR文件；其次是KPOINTS方面，等效的kspacing是0.198。最后是ENCUT方面，原本的POTCAR中为269.532eV，而我修改为了500eV。理论上都是足够高的计算精度了。

      4. 孤立原子能量问题：这一部分计算的精度确实没有那么高，但是计算下来能量误差大约是0.03eV左右，应该也不是导致能量差别达到-0.47eV/atom的最终原因。

      # 附录1 EOS计算代码

      cat > INCAR.relax <<!
Global Parameters
  NPAR   = 8             (Max is no. nodes; don't set for hybrids)
  ISTART =  0            (Read existing wavefunction; if there)
  ICHARG =  2            (Non-self-consistent: GGA/LDA band structures)
  LREAL  = F             (Projection operators: automatic)
  ENCUT  =  500          (Cut-off energy for plane wave basis set, in eV) 
  PREC   =  Accurate     (Precision level)  
  LWAVE  = .FALSE.       (Write WAVECAR or not)
  LCHARG = .FALSE.       (Write CHGCAR or not) 
  ADDGRID= .TRUE.        (Increase grid; helps GGA convergence) 
  ISPIN = 2              (spin-polarized calculations are performed.)
  # LVTOT  = .TRUE.      (Write total electrostatic potential into LOCPOT or not)
  # LVHAR  = .TRUE.      (Write ionic + Hartree electrostatic potential into LOCPOT or not)
  # NELECT =             (No. of electrons: charged cells; be careful)
  # LPLANE = .TRUE.      (Real space distribution; supercells)

Electronic Relaxation
  ISMEAR =  1            (Gaussian smearing; metals:1)
  SIGMA  =  0.20         (Smearing value in eV; metals:0.2)
  NELM   =  60           (Max electronic SCF steps)  
  NELMIN =  6            (Min electronic SCF steps)
  EDIFF  =  1E-08        (SCF energy convergence; in eV) 
  # GGA  =  PE           (PBEsol exchange-correlation)

Ionic Relaxation
  NSW    =  60           (Max electronic SCF steps)
  IBRION =  2            (Algorithm: 0-MD; 1-Quasi-New; 2-CG)
  ISIF   =  4            (Stress/relaxation: 2-Ions, 3-Shape/Ions/V, 4-Shape/Ions)
  EDIFFG = -0.001        (Ionic convergence; eV/AA)
  # ISYM =  2            (Symmetry: 0=none; 2=GGA; 3=hybrids)
!

cat > INCAR.static <<!
Global Parameters
  NPAR   = 8             (Max is no. nodes; don't set for hybrids)
  ISTART =  0            (Read existing wavefunction; if there)
  ICHARG =  2            (Non-self-consistent: GGA/LDA band structures)
  LREAL  = F             (Projection operators: automatic)
  ENCUT  =  500          (Cut-off energy for plane wave basis set, in eV) 
  PREC   =  Accurate     (Precision level)  
  LWAVE  = .FALSE.       (Write WAVECAR or not)
  LCHARG = .FALSE.       (Write CHGCAR or not) 
  ADDGRID= .TRUE.        (Increase grid; helps GGA convergence) 
  ISPIN = 2              (spin-polarized calculations are performed.)
  # LVTOT  = .TRUE.      (Write total electrostatic potential into LOCPOT or not)
  # LVHAR  = .TRUE.      (Write ionic + Hartree electrostatic potential into LOCPOT or not)
  # NELECT =             (No. of electrons: charged cells; be careful)
  # LPLANE = .TRUE.      (Real space distribution; supercells)
  # GGA   = PE

Static Calculation
  ISMEAR = -5            (tetrahedron method for DOS) 
  # LORBIT =  11         (PAW radii for projected DOS)
  # NEDOS  =  2001       (DOSCAR points) 
  NELM   =  60           (Max electronic SCF steps)
  EDIFF  =  1E-08        (SCF energy convergence; in eV)
  EDIFFG = -0.001
!

cat > KPOINTS <<!
Automatic generation
0
Monkhorst-pack
9  9  9
0  0  0
!

echo 'a0' 'volume' 'free_energy(eV)' >ev.out

for i in $(seq 3.40 0.01 3.60)
do

cat > POSCAR <<!
Ni4 fcc
1.0000000000
    $i                  0.0000000000        0.0000000000
    0.0000000000        $i                  0.0000000000
    0.0000000000        0.0000000000        $i
Ni
4
Direct
    0.0000000000        0.0000000000        0.0000000000
    0.5000000000        0.0000000000        0.5000000000
    0.0000000000        0.5000000000        0.5000000000
    0.5000000000        0.5000000000        0.0000000000
!

#优化计算
cp INCAR.relax INCAR
echo "a=$i"; time mpirun -n 64  ~/vasp.6.4.1/bin/vasp_std
cp CONTCAR POSCAR
rm INCAR

#静态计算
cp INCAR.static INCAR
echo "a=$i"; time mpirun -n 64  ~/vasp.6.4.1/bin/vasp_std
V=$(grep "volume" OUTCAR | tail -1 | awk '{printf "%12.9f \n", $5 }')
E=$(grep "TOTEN" OUTCAR | tail -1 | awk '{printf "%12.9f \n", $5 }')
echo $i $V $E >> ev.out
rm INCAR
done

      # 附录2 孤立原子能量计算

      cat > INCAR <<!
Global Parameters
  NPAR   = 8             (Max is no. nodes; don't set for hybrids)
  ISTART =  0            (Read existing wavefunction; if there)
  ICHARG =  2            (Non-self-consistent: GGA/LDA band structures)
  LREAL  = F             (Projection operators: automatic)
  ENCUT  =  500          (Cut-off energy for plane wave basis set, in eV) 
  PREC   =  Accurate     (Precision level)  
  LWAVE  = .FALSE.       (Write WAVECAR or not)
  LCHARG = .FALSE.       (Write CHGCAR or not) 
  ADDGRID= .TRUE.        (Increase grid; helps GGA convergence) 
  ISPIN = 2              (spin-polarized calculations are performed.)
  MAGMOM= 1*2            (Initial mag, result of statis cal)
  NELMDL  = 5            (the number of non-self-consistent steps at the beginning)

Atom Calculation
  IBRION = -1            (no update)
  NSW = 0                (no update)
  ISIF = 2               (force calculation)
  ISYM = 0               (no sys)
  ISMEAR = 0             (Gauss method for DOS) 
  SIGMA = 0.05           (atom calculation)
  NELM   =  100          (Max electronic SCF steps)
  EDIFF  =  1E-06        (SCF energy convergence; in eV)
!

cat > KPOINTS <<!
Monkhorst Pack
0
Monkhorst Pack
 1  1  1
 0  0  0
!

cat > POSCAR <<!
Ni atom
1.0000000000
    15.000000000        0.0000000000        0.0000000000
    0.0000000000        15.001000000        0.0000000000
    0.0000000000        0.0000000000        15.002000000
Ni
1
cart
 0.0000000000        0.0000000000        0.0000000000
!

echo 'volume' 'free_energy(eV)' >ev.out

time mpirun -n 64  ~/vasp.6.4.1/bin/vasp_std
V=$(grep "volume" OUTCAR | tail -1 | awk '{printf "%12.9f \n", $5 }')
E=$(grep "energy without entropy" OUTCAR | tail -1 | awk '{printf "%12.9f \n", $5 }')
echo $V $E >> ev.out
