# 1.1 弹性模量的测试

对于两组结果分别进行线性回归分析，分析结果如下：

对于ENCUT=267.882计算，C11=433.894284Gpa，回归R2=0.9989

对于ENCUT=367.882计算，C11=440.171799Gpa，回归R2=0.9994

补充，即使把图中的奇异点去掉，对于弹性常数的计算结果也几乎没有影响（因为总共有100个数据点，奇异点只有2-4个）

简而言之，如果增大ENCUT，似乎可以得到一个线性效果更好的结果，但是因为对于Fe的FCC构型没有实验，因此难以分析是否计算的弹性模量更加准确。
