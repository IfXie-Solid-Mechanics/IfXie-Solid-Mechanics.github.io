          # VASP五元合金的HCP构型整合工作流

          主要工具为ATAT软件，sqs2poscar插件和atomsk插件。

          # 1. 确定生成的dataset

          本次工作的目标势函数是FeNiCrCoPd和FeNiCrCoMn的势函数，因此总计2种元素组合的五元合金需要被建模；

          每种合金的原子比例总仅设计1档，即20:20:20:20:20；

          对于晶格常数的设置，由于HCP是特殊结构，因此设置一个初始的晶格常数，然后做结构优化。（这里需要注意HCP结构优化后是否可能最终得到FCC构型，但是理论上是有困难的，因为虽然FCC构型更加稳定，但是从HCP构型回到FCC构型同样需要翻越势垒）

          然后对于结构优化的结果，还可以进行AIMD扩充数据集。（同理，需要注意HCP结构优化后是否可能最终得到FCC构型）

          # 3. 使用VASP对于上述构型进行结构优化处理

# Ncore = 64

# the end cut

# the algo's choice
ALGO = Fast

# the kpoint set
KSPACING = 0.2

# 1.3 E_cut of Mn
ENCUT = 350.8232

# not create WAVECAR CHGCAR

          # 4. 批量上传

          将构造构型的命令进行调整

          #!/bin/bash
sqs2poscar bestsqs.out
num=1
for i in $(seq 2.05 0.05 5.00);
do
    mkdir CrFeCoNiPd_$(printf "%03d" $num)
    cd CrFeCoNiPd_$(printf "%03d" $num)
    cp ../bestsqs.out-POSCAR .
    sed -i 2s/.*/${i}/ bestsqs.out-POSCAR
    atomsk bestsqs.out-POSCAR -sort species up -frac POSCAR
    cd ..
    num=$(($num+1))
done
