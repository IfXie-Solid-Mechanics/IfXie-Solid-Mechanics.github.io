创建CrFeCoNiPd目录，并进入该目录。

初始完美晶格的生成采用Atomsk，元素的话先随便写了一个Ni，命令如下：

atomsk --create fcc 3.53 Ni orient [-110] [001] [110] -dup 2 2 5 POSCAR

# 1.2 用VASP获得准随机高熵合金构型

这一步由2个部分组成：打乱元素序列得到新的构型、用VASP做结构优化。

打乱元素顺序的bash脚本示例如下所示：

#!/bin/bash

# 输入文件名
input_file="POSCAR"
# 输出文件的前缀
output_prefix="POSCAR"
# 用于记录生成的哈希值
declare -A generated_hashes

# 获取POSCAR文件的前8行
head -n 8 "$input_file" > header.txt

# 获取POSCAR文件的第9到第48行
sed -n '9,48p' "$input_file" > body.txt

# 循环150次生成150个新的POSCAR文件
for i in {1..150}; do
    # 创建一个新的文件名
    output_file="${output_prefix}-$(printf "%03d" $i)"

    while true; do
        # 随机打乱body.txt的内容
        shuffled_body=$(shuf body.txt)

        # 计算打乱后的内容的哈希值
        hash_value=$(echo "$shuffled_body" | sha256sum | cut -d ' ' -f1)

        # 检查哈希值是否已存在
        if [[ -z ${generated_hashes[$hash_value]} ]]; then
            # 如果没有重复，保存该哈希值并退出循环
            generated_hashes[$hash_value]=1
            break
        fi
    done

    # 将前8行和打乱后的内容合并，输出到新的文件
    echo -e "$(cat header.txt)\n$shuffled_body" > "$output_file"
done

# 删除临时文件
rm header.txt
rm body.txt

echo "生成完毕！"

将这些构型与POTCAR文件进行打包得到VASP的结构优化计算输入文件，脚本如下

#!/bin/bash

for i in {1..150}; do
    poscar_file=../POSCAR-$(printf "%03d" $i)
    mkdir CrFeCoNiPd_110_$(printf "%03d" $i)
    cd CrFeCoNiPd_110_$(printf "%03d" $i)
    mv $poscar_file ./POSCAR
    cp ~/05_atomsk/quinary_wuzhen_1/CrFeCoNiPd/CrFeCoNiPd_1_1/POTCAR .
    cd ..
done
