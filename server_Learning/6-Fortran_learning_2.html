          # Fortran学习笔记-2

          本部分主要简单介绍Fortran的流程控制

          # 1. 逻辑语句和逻辑运算

          # 1.1 逻辑类型

          逻辑类型：.true.，.false.。0被视作false，1被视作true

          logical var
var = (a>10)
          1
          2

          # 1.2 逻辑判断

          这个根据版本不同，符合有所不同：

          Fortran 77：.EQ.，.NE.，.GT.，.GE.，.LT.，.LE.。

          Fortran 90：==，/=，>，>=，<，<=。 注意不等号和c语言不同。

          # 1.3 逻辑运算

          优先级比上面的逻辑判断更低，所以一般刚需括号，.AND.，.OR.，.NOT.，.EQV.，.NEQV.

          if(a>=80 .and. a<90) then
    ! do something
end if

          # 6. 其他控制语句

          pause: 程序暂停，等待用户按下enter键后继续执行。

          stop: 程序直接退出。

          continue: 不大有用，相当于无效语句。

          goto: 强制跳转(垃圾语句)。

          cycle: 在循环中相当于c语言的continue。

          exit: 在循环中相当于c语言的break。
