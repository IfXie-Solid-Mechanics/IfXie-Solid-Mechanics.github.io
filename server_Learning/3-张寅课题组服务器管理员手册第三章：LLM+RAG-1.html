        # 2. 文档格式转化

        由于我们的知识库的文件可能是各种各样的，比如论文的pdf文件，比如从网页上爬下来的文本文件，比如一些脚本程序文件。

        在这一步，我们需要先统一转化为json文件，然后再将json文件统一转化为规则txt文件，以便于后续的操作。

        # 2.1 确定目标 JSON 结构

        首先，明确 JSON 文件的格式和内容。通常会包括以下字段：

        id: 唯一标识，便于文件管理；

        content: 文本内容，主要核心部分；

        metadata: 元数据（如文件名、页码等）

        例如：

        [
    {
    "id": "doc_0000001",
    "content": "This is the content of the document.",
    "metadata": {
        "source": "1-s2.0-S0022509624001066-main.pdf",
        "page": 1
        }
    }
    ...
]

        # 2.2 使用PyPDF2进行格式转化

        先安装库环境：

        pip install PyPDF2

        # (a) 对于 PDF 文件

        使用PyPDF2提取文本内容，示例脚本：

        from PyPDF2 import PdfReader

def convert_pdf_to_json(pdf_path, json_file):
    reader = PdfReader(pdf_path)
    data = []
    
    for page_num, page in enumerate(reader.pages, 1):
        text = page.extract_text()
        data.append({
            "id": f"pdf_page_{page_num}",
            "content": text,
            "metadata": {
                "source": pdf_path.split("/")[-1],
                "page": page_num
            }
        })
    
    # 写入 JSON 文件
    import json
    with open(json_file, 'w', encoding='utf-8') as f:
        json.dump(data, f, ensure_ascii=False, indent=2)

pdf_path = "test/1-s2.0-S0022509624001066-main.pdf"
json_output = "output.json"

convert_pdf_to_json(pdf_path, json_output)
