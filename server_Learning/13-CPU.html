              # CPU功能性简介

              主要基于我在并行计算上学习到的相关知识

              # 1. CPU 功能性名词解释

              1. 指令执行：读取、解码和执行来自操作系统和应用程序的指令；

              2. 数据处理：指对数据进行算数运算或逻辑运算；

              3. 控制协调：协调和管理计算机系统中其他硬件组件的工作。

              # 2. CPU核心

              先看看几个我们即将解释的与CPU核心相关的专有名词：物理CPU (physical CPU)、 逻辑CPU(logical CPU)、虚拟CPU(virtual CPU)、核数(core)、线程数(thread)、颗数(socket)

              1. 物理CPU (physical CPU)：这个概念需要回溯到奔腾(Pentium)处理器的年代，那个时代还没有逻辑CPU(logical CPU)、虚拟CPU(virtual CPU)等概念。绝大多数计算机主板上仅仅有安装了一块CPU，但是少数的一些企业级计算机或者是大型的服务器因为对更大处理性能的需要才会在同一块主板上存在两块或更多的CPU芯片。因为这些CPU芯片与主板上的其他组件通过一个连接器（connector）或者 插槽（socket）通信，所以此时计数CPU个数就很简单了：主板上存在多少个这样的连接器（connector）或 插槽（socket）,那么这台计算机就能有多少个cpu

              2. 超线程技术（HyperThreading）：上述的多个物理CPU因为相互之间的通讯必须走系统总线（the system bus），而系统总线通常速度都很慢。这样就常常会造成性能瓶颈，无法充分利用每一个CPU提供的计算能力。所以在一颗物理CPU来执行多个线程或者进程，就可以避免上述这个通讯瓶颈。因此逻辑CPU(logical CPU)、虚拟CPU(virtual CPU)等概念开始逐步被提出。

              3. 逻辑CPU(logical CPU)：随着超线程技术同时被提出的一个概念，在一棵物理CPU最多同时执行2个线程则说明这个物理CPU上有两个逻辑CPU。超线程工作原理、逻辑CPU、物理CPU的示意图如下图所示。从图中我们可以看出，两个逻辑CPU共用一个ALU（算数逻辑单元），所以此时还并不是真正的并行。

              1. 核数(core)：虽然超线程技术的确提供了更高的性能上限，但是终究无法像两个独立的物理处理器那样。所以进一步的改进（只是改进！超线程技术仍然被保留）是将所有处理器组件做的更小，实现在同一个芯片上可以同时存在多个处理器。在同一个芯片上的每一个处理器我们称之为核（Core）。因为每一个核都具有了独立的CPU功能，因此物理CPU的定义在此被更新：物理CPU数=核数。下面这张图展示了一个芯片上的4核，8逻辑CPU的架构。CPU之间的通讯改为了BUS INTERFACE通信。

              1. 线程(thread)：总线程数=逻辑CPU总数，比如上图框架可以说是一个4核，8线程架构芯片。

              2. 虚拟CPU(virtual CPU):基本等同于逻辑CPU，但是存在一点不同：虚拟CPU更多的是限定在计算虚机化的语境内。

              目前在个人向计算机的一块主板上，往往只有一个CPU芯片。商家主要标注芯片上的核数、线程数。（超线程技术往往是2倍的超，但是也有的架构是4倍的超）

              # 3. 其他CPU性能参数

              1. 主频：CPU每秒钟可以执行的时钟周期数。但是需要注意主频不完全等于性能，不同架构对于CPU的影响比主频还大：比如i5-12400F的主频是2.5GHz，其性能明显强于拥有4.0GHz主频的i7-6700k。所以主频一般是在同架构、同代下进行对比。

              2. 缓存：CPU内部的高速存储器，速度远高于内存（RAM）。缓存越大，往往CPU就能减少访问内存的次数，那性能也会得到提升。

              3. 热设计功耗(TDP)：表示CPU在满载运行时，预计产生的最大热量。对于台式机，其是选择电源散热器的重要依据；对于笔记本，其是续航的重要指标。

              4. 集成显卡：把显卡集成到CPU内部，这样无需独立显卡也可以执行一些正常的显卡功能。

              # 4. Intel的CPU简写说明

              酷睿系列(CORE)：i3，入门级，适合办公、上网；i5，中段主流，性能稍强，中轻度的游戏性能、内容创作性能，性价比不错；i7，性能更强劲，中重度的游戏性能、内容创作性能。i9，极致性能，常见于发烧级游戏，搭建工作站。

              酷睿后缀：K表示解锁超频技术（散热条件允许时，解放主频，拥有更高性能）；F表示无集成显卡；KF就是前两者的结合（可超频且无集成显卡）；T表示功耗较低。

              # 5. AMD的CPU简写说明

              锐龙系列(RYZEN):RYZEN3，对标i3；RYZEN5，对标i5，但是性价比更高一点；RYZEN7，对标i7；RYZEN9，对标i9

              锐龙后缀：X表示主频更高的版本；X3D搭配了3D V-Cache缓存技术，游戏性能大幅提升；G表示搭载了较强性能的核显。

              # 6. 个人向总结

              对于理工科学生，i5/RYZEN5是入门标配，如果有能力上i7/RYZEN7的笔记本就足矣，更高的性能必要性不大。以及买新不买旧，在对应系列中挑选最新版本往往都是较优选择。
