            # 笔记本向显卡功能性简介

            # 1. GPU的发展历史

            显卡的前期发展历程：从CPU那里抢一些活来干。在最初的VGA(Video Graphic Array)时代，显卡的主要功能仅仅是输出图片，图形运算等功能仍然由CPU执行。但是当Windows操作系统提出后，工程师发现仅仅使用CPU来处理图形运算的功能已经很吃力了，所以开发出了一颗专用的芯片来处理图形运算，从而将CPU解放了出来，让Windows界面运行起来非常流畅。此后视频解码、硬件加速、3D渲染等任务都被显卡芯片来承担。

            GPU（Graphic Processing Unit，图形处理器）是由NVIDIA在1999年发布GeForce 256这一款划时代的产品时提出的。

            此后GPU的优化有很多，但是其中和科学计算联系紧密的是GPU远超CPU的浮点计算能力，在API的加持之下，GPU就可以执行更多的非图形指令。这推动了GPU正在朝着非图形领域、多元化方向发展。

            # 2. 笔记本显卡分类

            1. 独立显卡（独显）：性能之王，缺点就是稍微有些耗电、耗钱（第二个可能是我的缺点

            2. 核心显卡（核显）：就是把显卡集成到CPU内部，这种显卡的显存是通过占用内存的方式得到的，并且独立显卡显存频率是明显快于内存频率的，所以核心显卡不仅会占用一定内存，性能往往也低于独立显卡。

            3. 集成显卡：把显卡集成到了计算机主板上，性能介于核心显卡和独立显卡之间，价格也是如此。我认为这个产物更多是因为核心显卡已经满足不了新时代很多图像计算相关任务的需求，所以给出了一个核心显卡的进化版。（有的主机爱好者图吧垃圾佬在装机也会把这个显卡当作一个保险，因为一旦机器的独立显卡坏了要拿去维修时，用这个核心显卡也勉强能坚持这段维修时间）

            # 3. 显卡参数

            显卡架构：对于非专业人士，最好选择最新架构，比如NVIDIA目前的Blackwell架构。

            显存：对于笔记本电脑，显存是比较尴尬的。显存上得比较高对于笔记本功能完全是溢出的，但是想做一些AI相关的计算这点看似很高的显存又是不够的。（比如笔记本的5060的8G显存已经能够满足百分之百的需求了，而5090的24G显存对于一般用户而言意义不大。（除非真的很追求游戏超高画质或其他专业需求

            CUDA核心数：这个是GPU的一个关键参数，它和GPU的计算速度，程序并行率是显著相关的。（但是笔记本的显卡往往都是主机显卡的阉割版，笔记本的5090的CUDA核心数相比于主机版直接少了一半，所以科学计算速度最好还是考虑服务器端的显卡吧）

            # 4. NVIDIA独立显卡推荐

            注意，这里推荐的都是笔记本阉割版独立显卡，如果你有精力外接显卡坞那就去看桌面版的。

            RTX 5060: 8G显存较为充足，部署一个本地小模型来应付杂活足矣，性能方面流畅运行大部分游戏（不能开到极致画质，但是也很够看了），10分钟内视频剪辑十分流畅（允许加入很多特效），修图功能更是完全满足上限。

            RTX 5070: 8G显存，性能肯定提升了一点，单从CUDA核心数量上，提升幅度约为百分之十左右（说人话就是5060吃力的，5070也吃力；5070能玩的，5060也能玩）。但是参考华硕天选6 Pro，其他配置不变，这点显卡性能提升贵出了2000元。

            RTX 5070Ti: 12G显存，性能提升更加明显，属于有预算可冲。画质是能更上一层楼的，但是我没有在华硕天选6 Pro中看到这个配置提升的价格。如果比RTX 5060贵2500元才是我心里能接受的价位。

            RTX 5080, RTX 5090: 有钱真好！但是需要注意一点，有的厂商可能也会为了流量给真的移动版5080/5090，但是在其他硬件方面存在明显瑕疵。（尤其是主板稳定性这一块是很多厂商容易下黑手的地方，因为很难从参数上看出来）如果都有这种极致需求了，直接选择大厂才是最稳的。

            # 5. 和CPU的适配性注意

            在上一篇文章中，我们简单地介绍了Intel和AMD的CPU命名时的一些规则，其中和GPU较为相关的如下：

            Intel带有F后缀表示无核显，所以如果我们买的电脑是有独立显卡的情况下，往往就去选带F后缀的CPU即可。

            AMD带有G后缀表示有核显，所以如果我们买的电脑是有独立显卡的情况下，往往就去选不带G后缀的CPU即可。

            当然，最好还需要避免一些显卡和CPU性能存在明显代差的情况。因为笔记本工作是一个木桶效应，工作效果往往取决于短板。比如用i3的CPU去搭配5090就是很离谱的事情。
