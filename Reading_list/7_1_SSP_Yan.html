          # 金属电子气模型——Drude模型

          1897年汤姆逊发现电子，Drude据此建立了一个简单模型，即：把金属中的电子当作理想气体，而重量非常大的正电荷则是阻碍电子运动的散射中心。这种将金属中自由电子视为经典气体的模型，通常被称作Drude模型

          但是这一理论很快受到一个质疑，即金属自由电子带有负电，因此在电场中会被加速。但显然，金属电子在外电场中不可能被无限加速。因此，对于Drude模型需要不断增加近似和假设理论。这些假设和近似将是本章主要内容。

          # 1. 基本假设

          完整的Drude模型具有一下的假设与近似：

          1. 自由电子近似：忽略电子和正电荷之间的长程库仑作用，它们之间只有靠近时才会发生瞬时的碰撞或散射。

          2. 单电子近似：忽略不同电子之间的相互作用。

          3. 碰撞假设：碰撞是瞬时的，并且电子在受到瞬时的碰撞后，速度大小和方向随机分布，导致电子碰撞后动量的矢量平均为0。

          4. 弛豫时间：电子受到碰撞的几率用弛豫时间 τ 表示在任意一个微小时间间隔 dt ( dt 趋近于0)中，任一电子受到碰撞的概率为 dt/τ 。（也可以理解为全部电子中受碰撞的个数的比例）

          总而言之，Drude模型模型的电子的相互作用被忽略，原子核和其他电子也仅仅是在碰撞时才能影响电子运动。电子和原子核的碰撞也被称为散射。

          同时，从上述假设中，已经能够解释电子会被外电场无限加速的问题，这将在下面的3.1部分中讲解。

          # 2. 弛豫时间

          从最初的Drude模型的弛豫时间出发，我们能得到两个重要的结论：

          1. 从0时刻开始，到t时刻电子没有发生散射的几率 p(t)=e−tτ ，随时间指数衰减。

          2. 弛豫时间 τ 是电子两次散射之间的平均时间间隔。

          # 5. 检验Drude模型

          详细见《固体物理基础》（阎守胜），通过Drude模型可以直接计算出金属的霍尔系数、一个有关热导率 κ 的比值 κTσ0 。

          实验结论为：

          1. 只有一价金属的霍尔系数与该理论符合较好，其他类型的金属符合很差，甚至符号与理论结果相反。这说明Drude模型只适用于一价金属的分析。

          2. 实验上，比值 κTσ0 的确是常数（维德曼-弗兰兹定律），但是却不是Drude模型计算出来的结果。这也昭示着Drude模型的局限性。
