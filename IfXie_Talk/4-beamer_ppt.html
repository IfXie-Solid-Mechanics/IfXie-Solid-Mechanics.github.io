# Beamer制作学术ppt教程

适用情景：

  1. 十分熟悉latex的语法，或者希望学习latex的语法；

  2. 在ppt中需要插入大量公式；

  3. 不希望花过多时间在ppt排版方面的工作；

  4. 希望将PPT以图片的方式放置在个人网站上。

# 1. Overleaf编码

Overleaf是一个强大的LaTex的线上IDE，其支持的LaTex的Beamer编码。并且拥有教程（https://cn.overleaf.com/learn/latex/Beamer%23Creating_a_table_of_contents）

Overleaf上编码出的是pdf文件，可以直接下载。

# 1.1 Overleaf使用细节

  1. 长宽比：Overleaf的默认每页ppt的长宽比是4:3，为了适应大部分16:9的屏幕（如果使用4:3的长宽比在16:9的屏幕两侧有黑边，其实也属于可以接受的范围），可以自己在开头部分定义一个对应的长宽比。

# 2. python转化

通过上一步的制作，现在获得了一个main.pdf的文件（请改为这个文件名，或者修改下面的代码部分）

然后使用python脚本，将pdf转化为图片，然后再将这些图片插入进pptx文件，最终得到我们想要的ppt文件。

import os
from pdf2image import convert_from_path
from pptx import Presentation
from pptx.util import Inches
from datetime import date

def pdf_to_ppt(pdf_path, ppt_path):
    # 将PDF转换为图片
    images = convert_from_path(pdf_path,dpi=300)
    # 创建一个新的PPT
    prs = Presentation()

    # 设置PPT页面尺寸（4:3），可以自行调整参数
    slide_width = Inches(12.8)  # 页面宽度，可以自行调整参数
    slide_height = Inches(9.6)  # 页面高度，可以自行调整参数
    prs.slide_width = slide_width
    prs.slide_height = slide_height

    # 将每个图片添加到PPT中
    for image in images:
        # 计算图片的尺寸和位置，保持4:3的比例，可以自行调整参数
        image_width, image_height = image.size
        image_ratio = image_width / image_height
        slide_ratio = slide_width / slide_height

        if image_ratio > slide_ratio:
            # 图片宽度超过页面宽度
            width = slide_width
            height = width / image_ratio
            left = 0
            top = (slide_height - height) / 2
        else:
            # 图片高度超过页面高度
            height = slide_height
            width = height * image_ratio
            top = 0
            left = (slide_width - width) / 2

        slide_layout = prs.slide_layouts[6]  # 使用空白布局
        slide = prs.slides.add_slide(slide_layout)

        # 将图像保存为临时文件
        temp_image_path = "temp_image.png"
        image.save(temp_image_path)

        # 将临时文件的路径添加到PPT
        slide.shapes.add_picture(temp_image_path, left, top, width, height)

        # 删除临时文件
        os.remove(temp_image_path)

        # 设置图片为不可移动和编辑
        for shape in slide.shapes:
            if shape.is_placeholder:
                shape.placeholder_format.contained = False
                shape.placeholder_format.move_with_text = False
                shape.placeholder_format.wrap_format.text_wrapping = 3

    # 保存PPT
    prs.save(ppt_path)

# 调用函数进行转换
print('正在读取bemaer文件')
current_date = date.today().strftime("%Y-%m-%d")
pdf_path = './main.pdf'  # 输入PDF文件路径
ppt_path = './main-'+ current_date +'.pptx'  # 输出PPT文件路径
pdf_to_ppt(pdf_path, ppt_path)
print('转换完成，已导出'+ppt_path+'文件！')

# 2.1 python库安装

运行上面的python脚本，一般会缺少一些库文件，这一部分用pip install自行安装即可。

pip install pdf2image
pip install python-pptx

# 2.2 poppler插件安装

运行这个python程序时，需要一个名为poppler的插件，用于将pdf转化为图片。这个插件不推荐安装在Linux系统上（因为大概不会使用Linux做ppt吧），Windows的安装教程在网上有很多，这里简单地说明在macos上的安装方法：

brew install poppler
1
