        # 网站更新说明

        # 1. 右侧边栏优化

        由于之前对于右侧边栏的开发思路实现复杂度高：具体原因是优化后的版本，每多放一篇网页，就需要对于整个目录全部调整一次，这一方案显然是低效的。因此，在我还没有找到更好的右侧边栏实现方法之前，临时方案是删除右侧边栏的大纲，同时增大网页的可视空间。

        注：移动端的优化我直接用vuepress的自带插件摆烂的，因为我个人不常用移动端打开这一网站。

        # 2. 内容优化

        增加了“文献学习笔记”栏目，并且将杂项暂时先放入“科研错题本”中。

        # 3. 未来计划

        我能够感受到之前做的很多工作略显青涩，所以未来我更多的思路放在升级空间的成长性考虑（改用vuepress也是因为这一思想）

        可能会加入更多个性化元素。
