    # 势函数修正训练

    根据2025.06.06的121交流结果，讨论结果为势函数的训练过程仍然存在问题：

    1. 主要问题：存在“物理失真”的数据集。

    2. 次要问题：数据集增大，并将损失函数复杂化之后，应该适当增加训练步数。

    # 1. “物理失真”数据集分析

    目前检测出来的物理失真的数据集是：使用4个原子的纯金属构型跑AIMD的结果，这类结果在AIMD过程中，可能出现原子十分聚集的现象，而这一物理情景的内部电子轨道也会受到影响，再使用vasp的赝势在计算时容易出现失真的现象。

    解决的方案有两种：第一是修改损失函数的权重，对于这类失真数据集的权重进行下调；第二是直接将这类数据集进行删除。我们这次工作采用后者。

    那么删除的思路也有两种：第一是直接删除所有4原子纯金属的AIMD构型；第二种是针对计算结果能量较大的构型进行去除。后者在技术上更加麻烦，而且难以确定“能量较大”的定量描述。所以我们选择前者，尽管这可能会丢失部分合理的数据集，但是我们的数据集已经足够充分。

    # 1.1 具体修改

    由于我之前复杂但分类性较强的命名方法，可以直接在basic_dataset_CrFeCoNiPd目录中，通过下面命令删除所有Ni单质的AIMD构型（下划线很重要！否则会删除单质Ni的其他构型，甚至会删除含Ni合金）

    rm *_Ni_*
