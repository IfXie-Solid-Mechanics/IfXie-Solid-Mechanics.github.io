print('# the training data contains %d frames' % len(data_training))
print('# the validation data contains %d frames' % len(data_validation))

    # 2.Theory

    简而言之，直接输入的原子位置信息不具有平移不变性，置换对称性，旋转对称性等性质，因此需要做一个坐标变换。并且通过一个embedding神经网络，将rij的信息也融入到矩阵Di中。

    The notation of se_e2_a is short for the Deep Potential Smooth Edition (DeepPot-SE) constructed from all information (both angular and radial) of atomic configurations. The e2 stands for the embedding with two-atoms information. This descriptor was described in detail in the DeepPot-SE paper.

    对于"se_e2_a"这个Descriptor仅有两种性质，即平移不变性，置换对称性。

    # 2.1 Descriptor "se_e2_a"

    对于一个原子，使用一个Di矩阵进行信息描述：

    $ D^{i} = \frac{1}{N_{c}^2}(g^{i})^{T}R^{i}(R^{i})^{T}g^{*i}$

    1. Nc 是最大相邻原子数，对于所有原子都是相同的，如果原子截断半径内原子数超过这一数值，则会舍去远处原子，如果原子截断半径内原子数不足这一数值，则会在计算程序中自动补0。

    2. Ri∈RNc×4 是一个坐标位置信息矩阵，形式如下

      (Ri)j={s(rij),s(rij)xijrij,s(rij)yijrij,s(rij)zijrij}

      rij=rj−ri=(xij,yij,zij)

      s(r)是一个分段函数，但是二阶导数连续，其中有两个参数rc,rs。如果r<rc，则s(r)=1r，如果rc<r<rs ，则s(r)=1r(r3(−6r3+15r−10)+1),如果rs<r，则s(r)=0

      注：如果原子i周围的原子数小于Nc则会自动在R^{i}中补0。

    3. gi∈RNc×M 是嵌入矩阵

      $ (g^{i}){j} = N{e,2}(s(r_{ij})) $

      矩阵列数 M 的取值和神经网络节点设置有关，由用户设置参数决定

      Ne,2(s)是一个待训练的DNN

    4. g∗i∈RNc×M< 是取矩阵 gi 的前 M< 行，以减小计算量。

      $ M_{<} $的取值由用户设定参数决定。

    ·”learning_rate”: 设置学习率的类型及其衰减参数;

    ·”loss”:定义损失函数及其权重;

    ·”training”:包含训练数据、验证数据和训练设置。

    可以看到model内部由3个结构组成：type_map, descriptor, fitting_net

    1. type_map(元素映射):

      指定不同的原子类型。在此例中，"Li" 和 "Cl" 分别对应锂和氯.

    2. descriptor:

      定义描述符的参数，即选用什么训练模型，训练模型的参数怎么设置；

      "type" se_e2_a模型已经在上文进行了理论说明，一般较复杂的合金，化合物都可以使用这个模型；

      "sel" 这个设置了上文提到的Nc，两种原子的Nc都被设置为128；

      "rcut_smth", "rcut"分别指上文的rs,rc，单位为A；

      "neuron" 指的是网络节点+铺层，这里指铺层了3层神经网络，第一层20个节点，第二层40个节点，第三层80个节点，下文还有对于neuron进一步地讨论；

      "resnet_dt" 我暂时还没有看懂，查一查；

      "axis_neuron" 指的是上文提到的M<；

      "seed" 指的是初始指用种子1来生成。

    3. fitting_net

      定义拟合网络的结构及参数，用于拟合描述符得到的特征；

      "neuron" 指的是网络节点+铺层，这里指铺层了3层神经网络，第一层200个节点，第二层200个节点，第三层200个节点，下文还有对于neuron进一步地讨论；

      "resnet_dt" 我暂时还没有看懂，查一查；

      "seed" 指的是初始指用种子1来生成。

    可视化python脚本示例

    import numpy as np
import matplotlib.pyplot as plt

# 定义绘制散点图和对角线的函数
def plot(ax, data, key, xlabel, ylabel, min_val, max_val):
    data_key = f'data_{key}'
    pred_key = f'pred_{key}'
    ax.scatter(data[data_key], data[pred_key], label=key, s=6)
    ax.legend()
    ax.set_xlabel(xlabel)
    ax.set_ylabel(ylabel)
    ax.set_xlim(min_val, max_val)
    ax.set_ylim(min_val, max_val)
    ax.plot([min_val, max_val], [min_val, max_val], 'r', lw=1)

# 读取数据，并对e数据进行原子化处理
natom = 64
data_e = np.genfromtxt("./LiCl_DP_Tutorial_Example/chapter3/01.train/results.e.out", names=["data_e", "pred_e"])
data_f = np.genfromtxt("./LiCl_DP_Tutorial_Example/chapter3/01.train/results.f.out", names=["data_fx", "data_fy", "data_fz", "pred_fx", "pred_fy", "pred_fz"])

for col in ['data_e', 'pred_e']:
    data_e[col] /= natom

# 计算e和f的最小值和最大值
data_e_stacked = np.column_stack((data_e['data_e'], data_e['pred_e']))
data_f_stacked = np.column_stack((data_f['data_fx'], data_f['data_fy'], data_f['data_fz'], data_f['pred_fx'], data_f['pred_fy'], data_f['pred_fz']))

min_val_e, max_val_e = np.min(data_e_stacked), np.max(data_e_stacked)
min_val_f, max_val_f = np.min(data_f_stacked), np.max(data_f_stacked)

# 绘制散点图并保存结果
fig, axs = plt.subplots(1, 2, figsize=(12, 5))
plot(axs[0], data_e, 'e', 'DFT energy (eV/atom)', 'DP energy (eV/atom)', min_val_e, max_val_e)
for force_direction in ['fx', 'fy', 'fz']:
    plot(axs[1], data_f, force_direction, 'DFT force (eV/Å)', 'DP force (eV/Å)', min_val_f, max_val_f)
plt.savefig("dft_dp.png")
plt.show()
