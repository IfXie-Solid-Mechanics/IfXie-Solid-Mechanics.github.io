    # LAMMPS五元合金的单轴拉伸实验（测试阶段）

    基于分子动力学和原子间机器学习势函数对于上述合金构型进行拉伸实验；

    实验结果主要为：应力-应变曲线图，合金的弹性模量。

    # 1. LAMMPS单轴拉伸实验

    # 1.1 脚本学习

    这个单轴拉伸的Input脚本主要来自于师姐张靖雅，做了一些微调，下面逐步介绍。

    units       metal
boundary    p p p
atom_style  atomic

read_data   CrFeCoNiPd.lmp

pair_style	deepmd CrFeCoNiPd-compress.pb
pair_coeff  * *

# set temperature
variable    T equal 300.0
timestep    0.001

# ------------- relax on npt --------------
# initialize temperature
velocity    all create $T 161518 dist gaussian

# set ensemble
fix         1 all npt temp $T $T $(100*dt) iso 0.0 0.0 $(1000*dt)

# relax
thermo      2000
thermo_style custom step temp
dump        1 all custom 2000 rlx_$T_polycant.dump mass type xs ys zs
dump_modify 1 element Cr Fe Co Ni Pd sort id
run         30000
unfix       1
undump      1

# ----------- uniaxial tensile testing ------------------
variable    total_strn equal 0.3
variable    rate equal 1e9
variable    trans equal 1e12 # from s to ps
variable    valoferate equal ${rate}/${trans}
variable    valofrun equal ${total_strn}/${valoferate}/dt

# set the strain and the stress
variable    tmp equal lx
variable    l0 equal ${tmp}
variable    strnx equal "(lx - v_l0)/v_l0"
variable    strsx equal -pxx/10000

fix         2 all npt temp $T $T $(100*dt) y 0.0 0.0 $(1000*dt) z 0.0 0.0 $(1000*dt)
fix         3 all deform 200 x erate ${valoferate}
fix         def all print 100 "${strsx} ${strnx}" file stress-strain-x-polycant.dat screen no

thermo      1000
thermo_style custom step temp press v_strsx v_strnx lx
thermo_modify lost ignore
dump        1 all custom 1000 tens_$T_polycant.dump mass type xs ys zs
dump_modify 1 element Cr Fe Co Ni Pd sort id

run         ${valofrun}

    # 1.1.2 npt热浴

    基于300K的温度，使得每个原子被赋予相应的随机动能。

    velocity    all create $T 161518 dist gaussian
    1

    npt系综的建立，dt是timestep的默认变量。对于temp这个keyword，后面的3个变量分别是初始温度，最终温度，热浴步长。对于iso这个keyword，后面3个变量分别是，初始外加压强，最终外加压强，更压间隔。

    fix         1 all npt temp $T $T $(100*dt) iso 0.0 0.0 $(1000*dt)
    1

    输出控制

    thermo      2000
thermo_style custom step temp
dump        1 all custom 2000 rlx_$T_polycant.dump mass type xs ys zs

    # 1.1.3 单轴拉伸

    输入变量设置：总应变设置为0.3，应变率rate设为10的9次方每秒，转为每皮秒得到应变率变量valoferate，由总应变和应变率得到run的总步长变量valofrun；

    输出变量设置：lx在lammps中是box在x方向的长度，l0为初始长度，strnx为工程应变，pxx为box在x截面x方向上的应力（除以10000将应力单位转换为GPa）

    variable    total_strn equal 0.3
variable    rate equal 1e9
variable    trans equal 1e12 # from s to ps
variable    valoferate equal ${rate}/${trans}
variable    valofrun equal ${total_strn}/${valoferate}/dt

variable    tmp equal lx
variable    l0 equal ${tmp}
variable    strnx equal "(lx - v_l0)/v_l0"
variable    strsx equal -pxx/10000

    # 1.2 脚本运行

    首先，将我们原本的CrFeCoNiPd的预测relaxed构型更改为LAMMPS输入文件

    atomsk POSCAR CrFeCoNiPd.lmp
    1

    然后先在本地ZHANG01上进行尝试运行，为了防止崩溃，采用GPU的方式进行脚本运行

    conda activate deepmd
