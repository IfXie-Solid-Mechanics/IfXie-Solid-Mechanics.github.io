      # LAMMPS堆垛层错能测试

      基于分子动力学和原子间机器学习势函数对于单质或合金的fcc构型进行堆垛层错能测量实验；

      测量结果：对应构型的堆垛层错能、沿1/6[112]滑移的滑移过程的能量曲线。

      # 1. LAMMPS堆垛层错能测量

      # 1.1 脚本学习

      这个堆垛层错能的Input脚本主要来自于师姐张靖雅，做了一些微调，下面逐步介绍

      units		metal
boundary	p s p
atom_style	atomic

variable	dx equal sqrt(2)	# x: [110]
variable	dy equal sqrt(3)	# y: [111]
variable	dz equal sqrt(6)/2	# z: [112]

read_data	Ni-fcc_2.lmp

pair_style	deepmd CrFeCoNiPd-compress.pb
pair_coeff  * *

# ------------ set variable --------------
variable	xhalf equal (xhi+xlo)/2
variable	yhalf equal (yhi+ylo)/2
variable	zhalf equal (zhi+zlo)/2
variable	n equal 10
variable	dpstep equal 3.518316/sqrt(6)/10
variable	y1 equal ${yhalf}+5*${dy}*3.518316
variable	y2 equal ${yhalf}-5*${dy}*3.518316
# -------------select upper and lower layer------------------
region		upper1 plane ${xhalf} ${yhalf} ${zhalf} 0 1 0 units box
group		upper1 region upper1

region		calup plane ${xhalf} ${y1} ${zhalf} 0 -1 0 units box
region		callo plane ${xhalf} ${y2} ${zhalf} 0 1 0 units box
group		calup region calup
group		callo region callo
group		cal intersect calup callo

compute		Eperatom cal pe/atom
compute		Eatoms cal reduce sum c_Eperatom
# ----------- move atoms to form stacking fault -------------
thermo		1
thermo_style	custom step temp etotal pe ke pxx pyy pzz c_Eatoms

fix		2 all setforce NULL NULL 0.0
minimize	1e-15 1e-15 100000 100000
run		0

variable	E00 equal c_Eatoms
variable	E0 equal ${E00}
variable	E equal c_Eatoms
variable	deltaE equal (v_E-v_E0)/lx/lz*1.602*10000   # mJ/m2

variable	m loop $n
label		loop

displace_atoms	upper1 move 0.0 0.0 -${dpstep} units box
minimize	1e-15 1e-15 100000 100000
run		1
print		"$m ${deltaE}" append txt/gsfe.txt screen no

next		m
jump		SELF loop

      # 1.1.1 边界条件设置

      boundary	p s p
      1

      在y方向上设置了非周期的动态边界条件，即box在y方向具有非周期性，且会根据原子位置，在y方向上进行收缩包裹。

      在这一部分，首先定义了upper1，这个是待滑移部分的原子，也就是上半部分会被拿来滑移搓堆垛。

      然后通过第二部分，定义了cal区域，就是上一部分划分出来的中间10个周期厚度的原子，这部分是我们的计算部分。

      最后，我们定义了计算内容，第一行是求出每个原子的势能Eperatom，第二行是求出这个部分的总势能Eatoms

      # 2. 初始构型的设置

      首先通过atomsk进行Ni-fcc的建模，由于y方向是滑移面方向，且上下原子层数较高，因此建模时略微增加一定厚度。

      atomsk --create fcc 3.5183159685144951 Ni orient [1-10] [111] [-1-12] -dup 10 40 10 Ni-fcc.lmp
      1

      总计24000个原子。

      # 3. 运行命令

      nohup lmp -i in.sfe > output.log 2>&1 &
      1

      计算时长7分钟。

      但是由于现有版本没有见过Ni堆垛层错的具体形式，所以计算结果的物理意义较差，尤其是越接近hcp构型时。
