        # 1. 代码学习

        从张老师的GitHub中copy过来的完整代码如下所示：

        # Strain energy of High Entropy Alloy
# Yin Zhang
# Mar 28, 2019

############## Initialization
units			metal
boundary		p p p
atom_style		atomic
##############

############## Atom definition
lattice			fcc 3.556398 origin 0 0 0 orient x 1 0 0 orient y 0 1 0 orient z 0 0 1
region			mybox block 0 15 0 15 0 15 units lattice

create_box		5 mybox
create_atoms	1 box

region			1 block 6 9 6 9 6 9 units lattice
group			inner region 1
group			outer subtract all inner

############## Potential
pair_style      eam/alloy
pair_coeff      * * /home/yzhang951/HEA_MC/NiCoCr.lammps.eam Ni Co Cr Co Cr


############## Thermo

        修改部分：

        1. 对于晶格指向进行了修改，在张老师的nature文章中，是对于(1-11)面沿着[110]方向扫描，因此我们将x轴定义为[1-11]，将y轴定义为[110]，将z轴定义为[-112]；

        2. 换用机器学习原子间势函数；

        3. 使用type/ratio进行严格的粒子数定义

        Only 2 types allowed when not using semi-grand in fix atom/swap command (src/MC/fix_atom_swap.cpp:217)

        代码能够运行，但是MC-MD模拟也许更加具有物理意义。

        # 3. 思路创新

        基于复现张老师nature中的结果为目标时，我们使用MC的目标就是探索相空间的组合，同时进行npt系综下的模拟进行能量最小化。

        新的代码如下所示:

        ############## Initialization
units			metal
boundary		p p p
atom_style		atomic
##############

############## Atom definition
lattice			fcc 3.556398 origin 0 0 0 orient x 1 -1 1 orient y 1 1 0 orient z -1 1 2
region			mybox block 0 8 0 20 0 15 units lattice

create_box		5 mybox
create_atoms	1 box

region			1 block 0 15 0 15 0 15 units lattice
mass            1 52
mass            2 54.94
mass            3 55.85
mass            4 58.93
mass            5 58.69

timestep        0.001

set             type 1 type/ratio 2 0.2	2457
set             type 1 type/ratio 3 0.5 1286
set             type 1 type/ratio 4 0.5 3286
set             type 3 type/ratio 5 0.5 5286
##############

############## Potential
pair_style      deepmd CrMnFeCoNi-compress.pb
pair_coeff      * *
##############

############## Minimize
velocity        all create 300.0 161518 dist gaussian
fix             1 all npt temp 300.0 300.0 $(100*dt) iso 0.0 0.0 $(1000*dt)
thermo          2000
thermo_style    custom step temp
dump            1 all custom 2000 rlx_300_polycant.dump mass type xs ys zs
dump_modify     1 element Cr Mn Fe Co Ni
run             30000
undump          1

############## MC_MD
fix				2 all atom/swap 1000 10 2372 700.0 ke yes types 1 2
fix				3 all atom/swap 1000 10 2727 700.0 ke yes types 2 3
fix				4 all atom/swap 1000 10 2372 700.0 ke yes types 3 4
fix				5 all atom/swap 1000 10 2372 700.0 ke yes types 4 5

thermo			1000
thermo_style    custom step etotal pxx pyy pzz pxy pyz pxz
dump            1 all cfg 1000 *.cfg mass type xs ys zs
dump_modify     1 element Cr Mn Fe Co Ni
