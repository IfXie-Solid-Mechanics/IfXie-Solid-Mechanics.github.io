# 《最高的休息法》

作者：久贺谷亮，曾在日本完成博士生涯以前的学业，而后在美国耶鲁大学脑科学担任博士后，现位于美国洛杉矶南湾的TransHope Medical担任院长。

本书采用了一种轻小说的写作方式，通过作者在美国博后生涯初期的故事，介绍正念休息的重要性。

# 摘抄与感悟

# 正念的起源

英文：Mindfulness

据说正念起源于原始佛教。19世纪维多利亚时期，英国人访问斯里兰卡时接触到这一概念，随后将此传播至西方国家。可以说，正念是西方人将东方的思考和冥想改编成符合他们习惯的产物。因此，正念早已摒弃了原有的宗教性质，相对而言更加注重实用性。

正念在美国流行的原因：毕竟美国人从小就被赋予了必须成功的使命，只要活着就必须不停地鞭策自己。也就是说，美国文化把无所事事视为一种罪过，只要活着，就必须一直和通往成功之路的压力奋战。竞争是必然的，为了成功，必须战胜压力不可。 但是，如此拼命也差不多到了极限。美国人很清楚如何能快速完成工作，怎样有效地赚钱，却从没想过如何让自己停下来，就像一台只有油门、没有刹车的汽车。就在这时，很久以前就从东方引进的正念重新引起了大家的注意。不知道该怎么休息的美国人突然意识到“这就是他们需要的”，于是一窝蜂地开始研究，发展到了现在这个状态。

（即美国人将“正念”开始视为“休息法”）

# 正念的科学理论支持

耶鲁大学的Judson Brewer在2011年发表的论文中，以有十年冥想经验的人为对象，测试他们每次正念冥想时，内侧前额叶皮质和后扣带皮层的活动程度都比较低。而研究表明，这两个区域除了能管理记忆和情感以外，同时也负责管理预设模式网络(DMN)。

DMN简单地说是一种“大脑空转”，而可怕的是其占脑部能量消耗的六成到八成，是造成大脑疲劳的主要原因。并且DMN的过度运转还可能造成反刍思考等易造成消极情绪的思考。但是DMN的存在有着一定的合理性，因为许多时候我们会陷入对于未来和过去的思考，而此时需要DMN来保持当下自己的呼吸等基本生理行为的执行。

（本人后来简单地调研了DMN，可以做一个简单的类比来了解DMN，可以把“电脑后台自动更新日志文件”比做“人脑的DMN运行”，所以DMN的运转是有一定意义的，但是过度运转即能量消耗占比达到八成就是弊大于利了。而正念之所以能够有效地降低DMN的运行耗能，是因为相比于刷视频、玩游戏等接收信息量大的更容易触发更新指令的休息方式，正念这种休息方式更加专注于自身信息，可以缓解触发DMN的要求）

（本人后来还了解到人脑的可塑性是较强的，也许一些不好的习惯导致了大脑DMN消耗占比过高，但是只要尝试改正，并有意地培养如正念这样积极健康的习惯，人脑的DMN消耗可以得到缓解，提升专注能力）

# 正念的心法

  1. 不做任何评价和判断，主动地将注意力集中在当下的经验上。（大脑的所有疲劳和压力都来自于过去与未来：对过去的事情心有不甘，对未来的事情充满不安）（饮食冥想、步行冥想均源于这一心法）

  2. 慈悲心是一种从内心培养慈爱，即对人的爱和怜悯的方法。简单来说，就是在自己内心培养正面情绪的一种方法。（默念：希望你能避开危险，平平安安；希望你幸福，安心自在；希望你身体健康）

  3. 正念不仅能够改造大脑，还能改变人类对压力的感知方式。正念可以帮助我们创造一个不用理性抑制压力，而是理性与情感和谐共处的大脑内部状态。（我自己对此的理解是，正念让我们的整个人脑各个脑区处于平等的状态，更容易达成理性与感性的平衡）

  4. 平和心（默念：世间就是如此，这就是世界最真实的样子。）接受当下，并有助于促进内心平和。

  5. 不想输的情绪最耗费大脑能量，最容易让大脑精疲力竭。（这是一种违背平和心的典型例子，尤其是不想输的情绪极大的放大了失败所导致的焦虑情绪，用一种还没发生的失败结果论来困住当下的自己）

  6. 幸福有48%是遗传，所以“感恩”很重要。总之，感恩之情可以化解愤怒、恐惧、嫉妒等各种负面情绪。

# 正念的习惯

  1. 饮食（地中海地区饮食）：

    最好每天都摄入的食物：蔬菜、水果、坚果、豆类、薯类、全麦谷物、鱼类、特级初榨橄榄油、芝士、酸奶；

    建议适度摄取的食物：鸡肉、蛋；

    尽量避免摄取的食物：红肉。

    （我个人认为其中坚果、芝士、酸奶等脂肪占比过高的食物是我会避免的，我认为牛肉应该也至少可以算为适度摄取的范围）

  2. 一些提升活力的习惯：

    拥有随时切换开/关模式的仪式；

    接触大自然；（真的很有效，想起了“久在樊笼里，复得返自然”）

    接触美的事物；

    培养自己埋头做一件事情的能力；

    回老家看看。
