    # P1 减脂

    # 1. 减脂原理

    减脂原理只有三点，分别是“热量缺口原理”、“特定的碳蛋脂比例”、“碳水的日内分配”

    # 1.1 热量缺口原理

    热量输入：食物提供的热量输入；

    热量输出：食物热效应（食物的热量不会完全被吸收为人体热量）、基础代谢（静息时，人体维持存活所需能量）、活动消耗（非静息导致的热量消耗）

    所以在减脂时，热量输入比热量输出为8:10到9:10之间时，便可达到减脂的目的。而减少热量输入，增加热量输出，或者两者兼而有之，均可达成这一目的

    # 1.2 特定的碳蛋脂比例

    光看食物的热量而不看碳蛋脂比例，往往会导致营养失衡。

    首先先了解热量输入与碳蛋脂比例的计算公式如下（单位kcal）：

    热量输入=(碳水克数×4)+(蛋白质克数×4)+(脂肪克数×9)

    所以碳蛋脂可以决定热量，而同一热量可以由多种碳蛋脂比例组合而成，但是不规律的组合可能导致缺少碳水导致的一系列病理症状；

    # 1.3 碳水的日内分配

    碳水的日内分配主要和胰岛素的升降调节有关：

    1. 训练日：训练餐后多给碳水（升高胰岛素促进增肌），其他餐少给碳水（热量平衡）；

    2. 休息日：每餐大概均分碳水（稳定胰岛素）；

    # 2. 热量缺口的制造方法

    首先，计算是算不准的，因为基础代谢需要估算，但是经验公式对于个体差异极大，并且活动消耗的估算难度更大，再说食物的热量也难以准确计算。

    但是仍然是存在解决方法的，即通过经验化的饮食配额去寻找热量缺口：

    # 2.1 经验化饮食配额

    力量训练日（男）： $（3g-3.5g碳水， 1.5g蛋白质， 0.8g脂肪）\times 体重量(以kg计算) $，即70kg男性训练日，应该吃210g-245g碳水，105g蛋白质，56g脂肪；

    力量训练日（女）： $（2.5g-3g碳水， 1.5g蛋白质， 0.8g脂肪）\times 体重量(以kg计算) $

    不健身时： $（2g-2.5g碳水， 1.2g蛋白质， 0.8g脂肪）\times 体重量(以kg计算) $

    # 2.2 寻找热量缺口

    周期：两周，因为体重在两三天内变化因素和食物驻留排空、人体的水分、肌肉量和脂肪量（最后这个才是我们关心的）有关，体重波动可能是另外两项的影响。但是长期变化主要由肌肉量和脂肪量决定。

    调整：如果体重掉了并且自身状态稳定，说明这是一个存在缺口的热量比例，没有的话就逐渐下调碳水，但是注意碳水摄入最好不要低于2g/kg

    # 3. 碳水的日内分配方法

    这部分为对于胰岛素的具体介绍、碳水投放具体时间节点。

    # 3.1 胰岛素

    功能：转运血液里的葡萄糖、氨基酸等营养物质进入肌肉和脂肪组织；

    负面影响：无力量训练+高胰岛素=增加脂肪。因为在无力量训练时，营养物质补给的优先级是脂肪大于肌肉的

    正面影响：有力量训练+高胰岛素=增肌增脂。因为力量训练会“破坏”肌肉组织，此时胰岛素会更优先于将营养物质补给到肌肉组织，实现增肌。同时也会补给脂肪的消耗，蛋此时优先级大幅下降。

    产生：胰岛素产生量和摄入碳水量、蛋白质量呈正相关关系，其中碳水摄入量占主导。

    扬长避短：力量训练后的训练餐可以故意提高胰岛素来增肌，其他餐和休息日分摊碳水保持胰岛素的稳定水平，减少增加脂肪。

    # 3.2 碳水投放具体时间节点

    这里介绍健身时间在下午或晚上的版本。

    # 3.2.1 下午健身

    早餐：全天0.3左右的碳水、适量蛋白质、适量脂肪。由于一夜空腹血糖较低，此时补充血糖以补充精力。（推荐：家园食堂素面条 + 2煮鸡蛋 + 1低脂牛奶）

    午饭：少吃甚至不吃碳水、适量蛋白质、适量脂肪。用于平衡一整天的碳水蛋白质脂肪摄入量。（推荐：一份瘦肉炒菜、足量蔬菜、少量或无主食）

    训前餐：全天0.2左右的碳水、可吃可不吃蛋白质、不吃脂肪。提供温和血糖支持（我初中练习体育时，训练前也需要直接和葡萄糖溶液）（推荐：一瓶蔗糖饮料，如脉动、冰红茶、宝力矿等，在去健身房前半个小时开始喝）

    晚餐（训后餐）：全天0.5左右的碳水，高蛋白质、低脂肪。提供营养，升高胰岛素，合成肌肉，减少减脂期的肌肉净损失（推荐：二两米饭+两份瘦肉炒菜）

    # 3.2.2 晚上健身

    道理同上，这里只写方案和推荐

    早餐：全天0.2左右的碳水、适量蛋白质、适量脂肪。（推荐：家园食堂馒头 + 2煮鸡蛋 + 1低脂牛奶）

    午饭：全天0.2左右的碳水、适量蛋白质、适量脂肪。（推荐：一两米饭+一份瘦肉炒菜、足量蔬菜）

    晚餐（训前餐）：全天0.2左右的碳水、可吃可不吃蛋白质、适量脂肪。（推荐：食堂面包干）

    训后餐：全天0.4左右的碳水、高蛋白质、低脂肪。（推荐：二两米饭，两份瘦肉炒菜）

    # 3.2.3 非训练日

    主要目的是均分碳水。

    早餐：全天0.3左右的碳水、适量蛋白质、适量脂肪。（推荐：家园食堂素面条 + 2煮鸡蛋 + 1低脂牛奶）

    午餐：全天0.3左右的碳水、适量蛋白质、适量脂肪。（推荐：一两米饭、一份瘦肉炒菜+水果）

    晚餐：全天0.3左右的碳水、适量蛋白质、适量脂肪。（推荐：一两米饭、一份瘦肉炒菜+水果）

    # 4. 其他

    Q1:快碳和慢碳是什么？

    A1:碳水食物有两个属性，一个是碳水率，一个是碳水分解速度，而这两个属性决定了“吸收对应食物后，人体内部血糖指数（GI）“的变化。而快碳一般指碳水分解速度快的碳水食物，但有的黑话里指的是高GI碳水食物。一般在训后餐吃高GI碳水即可，其余餐均不用在意，常见高GI碳水有米饭、馒头、白面包、西瓜、烤红薯、烤土豆、速食八宝粥。

    Q2:炒菜的脂肪含量估计方法？

    A2:一般清炒的脂肪含量可以默认为5g以下，如果底部有油但是不算油腻的可以默认10g左右，如果是水煮肉片等油腻感较强的炒菜脂肪在20g左右。并且需要注意煎鸡蛋，炒茄子，炸蘑菇都属于脂肪含量较高的菜。一般默认不吃肥肉。其他的如油条、坚果、蛋挞更是脂肪刺客。

    Q3：减脂期的少吃多动的怪圈

    A3:少吃多动有点像一句不明不白的浆糊话，因为在减脂的前期，通过饮食制造热量缺口是完全足够的，辅以力量训练避免肌肉流失。而减脂后期靠有氧运动来制造热量缺口更为合算，因为此时再少吃就可能引起身体代谢问题。
